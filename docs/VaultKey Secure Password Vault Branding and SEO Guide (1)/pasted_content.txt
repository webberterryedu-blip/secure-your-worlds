Entendi perfeitamente o que você quer melhorar no **Dashboard** do VaultKey.

Atualmente, os cards mostram pouca informação contextual: só o nick/título, e-mail (às vezes repetido), senha mascarada, categoria e alguns ícones de ação. Isso deixa o usuário sem saber rapidamente **o que** aquela credencial representa (ex.: "essa é minha conta LinkedIn pessoal", "esse é o GitHub da empresa", "esse é o login do AWS production").

### Objetivo principal
Tornar cada **CredentialCard** mais informativo e autoexplicativo, destacando:

- O **serviço/plataforma** claro (LinkedIn, GitHub, AWS, etc.)
- O **username/nick** específico daquela conta
- O **e-mail** associado (se diferente do geral ou relevante)
- URL (quando existir), talvez como link clicável ou tooltip
- Manter o resto (senha oculta, ações, categoria, favorito, expiração se tiver)

Sem sobrecarregar o card visualmente — usar hierarquia de texto (título grande, subtítulos menores, ícones).

### Sugestão de layout aprimorado para o CredentialCard

Estrutura recomendada (de cima para baixo):

1. **Título principal** (grande, bold) → nick + serviço quando possível  
   Ex.: "mauro.lima" • LinkedIn  
   ou "webberterryedu-blip" • GitHub

2. **Subtítulo / Detalhe principal** (menor, muted) → e-mail ou username alternativo  
   Ex.: mauro.lima@educacao.am.gov.br  
   ou weber@company.com (se for diferente)

3. **URL** (se existir) → como texto clicável pequeno ou ícone + tooltip  
   Ex.: linkedin.com/in/mauro-lima

4. **Senha** (mascarada, com toggle eye + copy)

5. **Badge da categoria** (E-mails, Projetos/Dev, etc.) + ícone

6. **Ações** (favorito, edit, delete, copy email/url)

#### 3. Funções auxiliares (adicione em lib/utils ou no próprio componente)

```ts
function getServiceFromUrl(url?: string): string | null {
  if (!url) return null;
  try {
    const hostname = new URL(url).hostname.toLowerCase();
    if (hostname.includes("linkedin")) return "LinkedIn";
    if (hostname.includes("github")) return "GitHub";
    if (hostname.includes("aws.amazon")) return "AWS";
    if (hostname.includes("google")) return "Google";
    // ... adicione mais conforme necessário
    return null;
  } catch {
    return null;
  }
}

function formatUrl(url: string): string {
  try {
    const u = new URL(url);
    return u.hostname + (u.pathname.length > 1 ? u.pathname : "");
  } catch {
    return url;
  }
}
```
